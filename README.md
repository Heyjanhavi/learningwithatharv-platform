# LearningWithAtharv.com

## Overview

**LearningWithAtharv.com** is a resource-sharing platform designed for medical students preparing for competitive exams. The platform allows users to register using their basic details and gain access to organized study materials that can be downloaded instantly.

The goal of the project is to create a centralized hub where students from different medical streams can easily access curated learning resources such as notes, exam preparation materials, and reference content.

---

## Features

### User Registration

Users can sign up by providing:

* Full Name
* Email Address
* Phone Number
* Medical Stream (MBBS / BAMS / BDS / BHMS / Others)

This information helps categorize users and manage resource distribution efficiently.

---

### Structured Learning Resources

Study materials are organized into categorized folders for easier navigation, such as:

* **AIPGET Preparation**
* **Prepladder / Prep Colour Notes**
* **Modern Medicine Notes**
* **ECG Learning Resources**
* **Exam Questions & Practice Sets**
* **Additional Medical Reference Materials**

Each category contains downloadable study materials relevant to that subject or exam.

---

### One-Click Download System

Users can download any resource with a **single click**, making the platform simple and efficient for quick access to study materials.

---

### Organized Folder System

Resources are stored and displayed in structured folders so users can quickly locate the material they need.

Example structure:

```
Resources
│
├── AIPGET
│   ├── Notes
│   ├── Practice Questions
│
├── Prep Colour Notes
│   ├── Subject Wise Notes
│
├── Modern Medicine
│   ├── Study Material
│
├── ECG
│   ├── ECG Interpretation Guides
│
└── Question Banks
    ├── Exam Practice Sets
```
